## Hexlet HTTP request
В этом курсе будут подробно рассмотрены концепции:
* Promise (Futures)
* Coroutines (using Generators)
* Async/Await

Дополнительно в курсе будет рассмотрен следующий набор тем:

* Формат данных json
* Итераторы
* Атаки в сети (CSRF)
* Модули nodejs: querystring, url, http
